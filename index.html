GitHub → Jenkins → Nginx → AWS EC2

About This Project

This is a simple DevOps pipeline demo that takes code from GitHub, builds it with Jenkins, and deploys automatically to an AWS EC2 instance running Nginx.

CI/CD Pipeline

  • 📂 Code → GitHub
  • ⚙️ Build → Jenkins
  • 📦 Deploy → Docker/Nginx
  • ☁️ Host → AWS EC2
